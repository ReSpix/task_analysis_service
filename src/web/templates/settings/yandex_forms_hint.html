Инструкции по настройке Яндекс Формы

Все настройки Яндекс Форм производятся в сервисе Яндекс Формы , в разделе "Интеграции" необходимой формы.

Для настройки интеграции необходимо создать "Запрос с заданым методом":

  • URL: https://[IP адрес сервера]:[портсервера]/receive/{Ответы на все вопросы}
  • Метод: GET
  • Тело запроса: пусто
  • Заголовки: пусто

Переменную "Ответы на все вопросы" необходимо настроить следующим образом:

  • Вопросы: выбрать все
  • Фильтры: пусто
  • Форматирование ответа: JSON
  • Только вопросы с ответами: нет
